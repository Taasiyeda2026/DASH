# דשבורד פעילויות

## סקירה כללית
דשבורד פעילויות ומערכת תזמון בעברית. זהו יישום אינטרנט סטטי לחלוטין (HTML, CSS, JavaScript) ללא צורך בשלב בנייה או צד שרת מורכב.

## ארכיטקטורת פרויקט
- **סוג**: אתר סטטי
- **שפה**: HTML/CSS/JavaScript
- **שרת**: `server.js` — שרת קבצים סטטי מינימלי ב-Node.js על פורט 5000
- **נתונים**: קבצי JSON המאוחסנים בספריית `data/`

### קבצים מרכזיים
- `index.html` — דף הדשבורד הראשי עם כניסה למערכת
- `app.js` — לוגיקת כניסה וטעינת נתונים
- `dashboard.js` — רינדור הדשבורד ולוגיקה עסקית
- `style.css` — עיצוב משותף
- `server.js` — שרת קבצים סטטי עבור Replit
